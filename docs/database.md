# AzariaOS Database

Database:
PostgreSQL

ORM:
Prisma


## Models


### Profile

Stores personal information.

Fields:

- name
- title
- bio
- social links


### Skill

Stores technical skills.

Examples:

- JavaScript
- Node.js
- PostgreSQL


### Project

Stores portfolio projects.

Includes:

- description
- GitHub link
- demo link
- images


### Experience

Stores professional experience.


### Education

Stores academic background.


### Certificate

Stores certifications.
